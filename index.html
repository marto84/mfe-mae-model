MFE = Max Favorable Excursion (best upside reached within the lookforward window). MAE = Max Adverse Excursion (worst downside reached). Conditional probabilities filter by occurrences that actually reached the current slider level, then calculate what fraction went further. This is a base rate distribution, not a signal -- it frames risk, it does not predict direction.
